HighscoreManager

Das Programm "HighscoreManager" ist eine einfache Winforms Applikation mit einem GUI Fenster. Hier können Username, Spielname, Highscore, Datum und Kommentare angegeben werden. Sobald auf den Button "Save" geklickt wird, werden, falls vollständig und korrekt, die eingegebenen Daten in deinem DataGrid angezeigt. Gleichzeitig werden die gespeicherten Daten in einer csv. Datei ("Highscores.txt") auf dem Desktop gespeichert. 

Wenn das Programm neu gestartet wird, werden alle Daten die im Highscores.csv sind, im DataGrid angezeigt.
Außerdem werden die bereits gespeicherten Spielnamen in einer Combobox dargstellt.
